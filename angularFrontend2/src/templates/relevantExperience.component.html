Relevant Experience

For the Instagram-inspired features of Megagram, I employed a variety of tools to develop both the frontend and backend, enabling user authentication (login/registration/password-resets), user search, profile management, AI-powered chatbot messaging, peer-to-peer messaging, following/blocking users, & dynamic interactions on the home screen, including liking, commenting, and tagging in posts.

In addition, I drew inspiration from Amazon to create another suite of features within Megagram. This includes a personalized home page that displays tailored ads, product promotions, and enables users to update delivery addresses and search for items by category. I also implemented fully functional shopping cart and checkout pages, integrating seamless frontend and backend systems to enhance the user experience.
